# cargo-binstall

KernRift is **not** a Rust crate and is not published on crates.io.
`cargo-binstall` does not apply to this project.

## Installing KernRift

Use one of the following methods:

### Shell installer (Linux / macOS)
```sh
curl -sSf https://raw.githubusercontent.com/Pantelis23/KernRift/main/install.sh | sh
```

### Homebrew (macOS / Linux)
```sh
brew install kernrift
```

### Scoop (Windows)
```powershell
scoop bucket add kernrift https://github.com/Pantelis23/KernRift
scoop install kernrift
```

### AUR (Arch Linux)
```sh
yay -S kernrift
```

### Winget (Windows)
```powershell
winget install Pantelis23.KernRift
```

### Manual download
Prebuilt binaries for all platforms are available at:
https://github.com/Pantelis23/KernRift/releases
